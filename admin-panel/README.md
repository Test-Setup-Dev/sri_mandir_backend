## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## Deploy on Netlify

This admin panel is configured for static export and can be deployed from the repository root using the included [`netlify.toml`](../netlify.toml).

Use these settings in Netlify:

```text
Base directory: admin-panel
Build command: npm run build
Publish directory: out
```

Add this environment variable in Netlify before deploying:

```text
NEXT_PUBLIC_API_URL=https://your-backend-domain/api/
```

If your Laravel backend is hosted separately, make sure its CORS settings allow requests from your Netlify site domain.
